Configuration

Basic Settings

Home Page

Set your preferred home page URL that loads when the app starts.

Settings → Home Page → Enter URL

Zoom Settings

Enable or disable zoom functionality for better accessibility.

Settings → Display → Zoom Controls

Screen Rotation

Control how the app responds to device rotation.

Settings → Display → Screen Rotation

Security

Privacy Features

No Data Collection

MDWebView does not collect, store, or transmit any personal data or browsing history.

Local Storage

All settings and preferences are stored locally on your device using secure SharedPreferences.

No Network Requests

The app itself makes no network requests except for the websites you choose to visit.

Important Security Note

While MDWebView itself doesn't collect data, the websites you visit may collect information according to their privacy policies. Always review website privacy policies before providing personal information.

Troubleshooting

Common Issues

App won't load websites

Check your internet connection and try again.

  • Verify you have an active internet connection
  • Check if the website URL is correct
  • Try loading a different website

Settings not saving

Ensure the app has proper permissions:

  • Go to Android Settings → Apps → MDWebView
  • Check that storage permissions are enabled
  • Restart the app after changing permissions

Cookies not working

Check your cookie settings:

  • Go to Settings → Cookie Management
  • Ensure cookies are enabled for the websites you need
  • Try clearing existing cookies and reload the page

Performance Tips

  • Clear browser cache regularly for better performance
  • Close unused tabs to free up memory
  • Use dark mode to reduce battery consumption
